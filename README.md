# 💈 BarberiaCjev4 - Sistema de Gestión de Barbería

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![NetBeans](https://img.shields.io/badge/NetBeans-1B6AC6?style=for-the-badge&logo=apache-netbeans-ide&logoColor=white)
![Git](https://img.shields.io/badge/GIT-E44C30?style=for-the-badge&logo=git&logoColor=white)

Este proyecto es una aplicación de escritorio diseñada para la **automatización y administración eficiente** de servicios de barbería. Desarrollado como parte de la formación en **Ingeniería de Sistemas**, enfocado en principios de programación avanzada y estructuras de datos robustas.

---

## 🚀 Características Principales

* **📅 Gestión de Citas:** Interfaz intuitiva para agendar y organizar turnos de clientes.
* **👥 Registro de Usuarios:** Base de datos para el almacenamiento seguro de información de clientes y servicios.
* **💻 Interfaz Gráfica:** Entorno desarrollado en Java Swing para una experiencia de usuario fluida.
* **🏗️ Arquitectura Escalable:** Separación de lógica de negocio y presentación para facilitar el mantenimiento.

## 🛠️ Stack Tecnológico

* **Lenguaje:** Java SE
* **IDE:** Apache NetBeans
* **Gestión de Versiones:** Git & GitHub
* **Metodología:** Desarrollo basado en Ingeniería de Software

## 📂 Estructura del Proyecto

El proyecto está organizado siguiendo estándares de desarrollo limpio:
* `src/`: Código fuente principal (Lógica y UI).
* `nbproject/`: Archivos de configuración del entorno de desarrollo.
* `test/`: Pruebas unitarias (si aplica).
